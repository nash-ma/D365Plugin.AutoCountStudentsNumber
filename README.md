# ■D365Plugin.autoCountStudentsNumber
## ①学生が新規作成の時、「クラス名」が設定済の場合のみ、対象クラスの学生人数を計算してクラスの項目「学生人数」を更新する。　　　　　　　　　　
## ②「クラス名」が更新の時、変更前「クラス名」と変更後「クラス名」の値を取得し、それぞれの学生人数を計算して項目「学生人数」を更新する。
